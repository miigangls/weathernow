# WeatherNow

Aplicación web de clima en tiempo real por ciudad.

## Tecnologías
- Vite + React 18 + TypeScript
- TailwindCSS
- React Router v6+
- Jotai
- React Query (@tanstack/react-query)
- Axios

## Configuración
1. Copia `env.example` a `.env` y completa tu API key:

```
VITE_OWM_API_KEY=tu_api_key_de_openweathermap
VITE_OWM_BASE_URL=https://api.openweathermap.org
```

2. Instala dependencias y lanza el servidor de desarrollo:

```
npm install
npm run dev
```

## Estructura
- `src/app` Proveedores y router
- `src/components` Componentes UI y de dominio
- `src/features` Hooks por dominio: countries, geocoding, weather, favorites
- `src/lib` Instancia de Axios, helpers
- `src/pages` Páginas y vistas

## Notas
- Countries se cachea 7 días con React Query.
- Búsqueda de ciudades con debounce (300ms) y límite de 10.
- Favoritos y últimas búsquedas persisten en localStorage vía `atomWithStorage`.
